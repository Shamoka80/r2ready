## **2. Access Levels and Roles**

| **Role**                             | **Access Level Description**                                              | **Example Permissions**                      |
| ------------------------------------ | ------------------------------------------------------------------------- | -------------------------------------------- |
| **Administrator**                    | Full system control, document creation, deletion, user access management. | Edit all templates, approve revisions.       |
| **Compliance Manager / EHS Officer** | Write access to governance, EHS, FM, audit, and training docs.            | Create/review R2v3 compliance documentation. |
| **Data Protection Representative**   | Full access to data security policies and logs.                           | Manage data sanitization and incidents.      |
| **QA / Repair Technicians**          | Write access to testing logs and operational procedures.                  | Record inspection/test data.                 |
| **Supervisors / Team Leads**         | Approve operational records, read higher-level documents.                 | Verify throughput and shift reports.         |
| **Operators / Line Staff**           | Read-only access to relevant SOPs and EHS instructions.                   | View safety and process documents.           |
| **External Auditor / CB**            | Restricted read-only access for audit evidence.                           | View compliance and audit folders.           |
| **Consultant / Trainer**             | Temporary access to onboarding materials and readiness reports.           | View-only or co-edit training documents.     |
